# Diceware password generator

Diceware password generator in Python, in one tweet. Using "alernative Beale" dictionary.

```bash
git clone https://github.com/shamrin/diceware.git
cd diceware
./generate
```

For example:

```
$ ./generate
cleft cam synod lacy yr wok
```
